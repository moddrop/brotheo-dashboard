🎓 Teacher Theo's Ultimate CSS Mastery Guide! 🎓

🌈 COLOR WIZARDRY:
• Basic Magic: red, blue, green, yellow, purple, orange, pink
• Advanced Spells: hotpink, lime, gold, turquoise, violet, coral, crimson
• Rainbow Magic: background: linear-gradient(45deg, red, orange, yellow, green, blue, purple);
• 🏆 Challenge: Create a sunset gradient with 5 different colors!
📦 LAYOUT SUPERPOWERS:
• Direction Control: flex-direction: row (→), column (↓), row-reverse (←), column-reverse (↑)
• Position Magic: justify-content: center, flex-start, flex-end, space-between, space-around
• Spacing Spells: gap: 10px (close friends), 50px (normal), 150px (distant cousins)
• 🏆 Challenge: Make a perfect grid of cards in 2 rows and 2 columns!
✨ EFFECT ENCHANTMENTS:
• Growth Magic: transform: scale(1.5) - becomes bigger!, scale(0.5) - becomes smaller!
• Rotation Spells: transform: rotate(45deg) - slight turn, rotate(180deg) - upside down!
• Glow Effects: box-shadow: 0px 10px 30px red - red magical aura!
• 🏆 Challenge: Make a card that glows rainbow colors and spins 720 degrees!
🔷 SHAPE TRANSFORMATION:
• Circle Magic: border-radius: 50% (perfect circle), 25% (rounded square)
• Border Styles: solid (normal), dashed (- - -), dotted (• • •), double (== ==)
• Size Control: width/height: 100px (small), 200px (medium), 300px (large)
• 🏆 Challenge: Create a heart shape using only CSS border-radius!
