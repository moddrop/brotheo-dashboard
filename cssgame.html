🎨 3-Hour CSS Adventure Learning Tips! 🎨

🌈 COLOR MAGIC:
• Basic Colors: red, blue, green, yellow, purple, orange, pink
• Fun Colors: hotpink, lime, gold, turquoise, violet, coral
• Try: background: linear-gradient(45deg, red, blue);
• Challenge: Make a rainbow background!
📦 FLEXBOX POWER:
• flex-direction: row (side by side) or column (up and down)
• justify-content: center, flex-start, flex-end, space-between
• gap: 10px (close), 50px (normal), 100px (far apart)
• Challenge: Make cards stack in a column!
✨ COOL EFFECTS:
• transform: scale(1.5) - makes bigger!
• transform: rotate(45deg) - spins around!
• box-shadow: 10px 10px 20px blue - colored shadows!
• Challenge: Make a card that spins and glows!
🔷 AMAZING SHAPES:
• border-radius: 0px = square, 50% = circle
• border: 5px solid red - colored borders
• width and height: 100px, 200px - change sizes
• Challenge: Make a heart shape with CSS!
🏆 FINAL CHALLENGE: Can you make a card that changes from a blue square to a pink spinning circle when you hover over it? Try combining everything you learned!
